# Research Plan: Dashboard Grid Layout Library

## Main Question
What JavaScript/TypeScript library should we use for a drag-and-drop, snap-to-grid dashboard page builder in a Svelte 5 + Tailwind CSS 4 application?

## Requirements
- Configurable grid with adjustable column count, row height, spacing/gap
- Component snapping to grid cells
- Drag to rearrange components on the grid
- Resize components
- Group and align components
- Works with Svelte 5 (rune-based, not stores)
- Lightweight, actively maintained
- Free/open-source

## Subtopics to Investigate

1. **React-grid-layout / Svelte equivalents** — The most popular grid layout libraries and their Svelte wrappers (svelte-grid-layout, svelte-grid, etc.)
2. **Vue-grid-layout ecosystem** — Gridstack.js, vue-grid-layout, and their vanilla/agnostic counterparts
3. **Vanilla/agnostic grid layout libraries** — Muuri, Packery, Masonry, Grid.js, and any new entrants
4. **Svelte-specific dashboard/layout solutions** — Any Svelte-native dashboard builders, layout engines, or component systems
